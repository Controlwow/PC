## Информация о проекте
Интернет-каталог продукции

## Использоуемые технологии
- HTML
- CSS
- JavaScript
- PHP 7.3
- Apache 
- MySQL
- laravel

## Информация по установке
- cd domains
- git clone
- composer self-update
- composer install
- создать файл .env из файла .env.example
- создать базу данных с именем "PC" и перейти в неё
- импорт файла pc.sql 
- перезапуск сервера
